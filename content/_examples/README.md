# Examples

This folder contains small "golden examples" that demonstrate the desired writing style.

These files are not full lessons.

They are calibration examples for ChatGPT, Codex, and the human author.

Use them to understand:

- lesson voice;
- rhythm;
- analogy quality;
- how to move from intuition to formal math;
- how to make explanations feel different from normal textbook notes.
- exercise design cards;
- final exercise structure;
- weak vs stronger exercise design;
- exercise set progression.

Do not treat these examples as official curriculum content.

## Files

- `golden-lesson-slice-limites.md` - lesson voice calibration.
- `motivation-examples.md` - motivation and analogy calibration.
- `golden-exercise-design-card.md` - canonical exercise design card example.
- `golden-exercise-limites.md` - final exercise structure example.
- `bad-vs-better-exercise.md` - weak mechanical exercise improved into a training device.
- `golden-exercise-set.md` - exercise set progression example.
