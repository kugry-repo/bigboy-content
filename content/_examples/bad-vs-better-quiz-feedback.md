# Bad vs better quiz feedback

This is an example/reference file only. It is not real unit content.

## Bad feedback

Student choice:
- A. La limite vaut $0$.

Feedback:

```text
Incorrect. The correct answer is B.
```

Why it is weak:

- It does not name the misconception.
- It does not explain why the answer is tempting.
- It gives no next step.

## Better feedback

Student choice:
- A. La limite vaut $0$.

Feedback:

```text
This answer is tempting because direct substitution gives 0/0. But 0/0 is not a conclusion; it tells you the expression must be transformed before evaluating the limit.
```

Diagnostic signal:
- The student treats an indeterminate form as a numerical value.

What to remember:
- $0/0$ means "transform first", not "the limit is $0$."

Remediation:
- Review the mini-revue on indeterminate forms, then practice one factorization item before retrying the quiz.

## Strong French version for learner-facing feedback

Cette réponse est tentante parce que la substitution directe donne $0/0$. Mais $0/0$ n'est pas une conclusion : c'est un signal qui dit qu'il faut transformer l'expression avant d'évaluer la limite.
