## Objectif de la série

Construire une progression courte pour apprendre à choisir la bonne transformation devant une forme indéterminée.

## Logique de progression

- Skill ladder : recognition -> core skill -> trap recovery -> method choice -> exam pattern.
- Exercise roles : warm-up, core-skill, trap-recovery, method-choice, exam-pattern.
- Difficulty range : application-directe to probleme-type.
- Estimated time : 35 minutes.
- Prerequisites : reconnaître $\dfrac{0}{0}$, factoriser un polynôme simple, utiliser une quantité conjuguée.
- Revision value : utile avant les exercices de continuité et de prolongement par continuité.

## Exercices

1. [[lc-ex-011]] — Reconnaître une forme indéterminée simple.
2. [[lc-ex-012]] — Factoriser pour simplifier un quotient.
3. [[lc-ex-014]] — Choisir la quantité conjuguée.
4. [[lc-ex-018]] — Prolonger une fonction par continuité après calcul de limite.

## Progression conseillée

Commencer sans calculatrice. Après chaque exercice, demander à l'élève de nommer le signal qui a déclenché la méthode.
