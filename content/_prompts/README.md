# Prompts

This folder contains reusable workflow instructions for Codex, ChatGPT, and the human author.

Prompts should be copied into Codex or ChatGPT and adapted to the target chapter before use.

These files are not generated lessons.

They describe repeatable authoring steps for chapter planning, mini-lesson drafting, exercise planning, review, and full chapter cleanup.

Mini-lessons are stored as separate files under each chapter `lessons/` folder.

Before using a prompt, replace placeholders such as `CHAPTER_INDEX_PATH`, `MINI_LESSON_FILE`, and `EXERCISE_ID_RANGE`.

Use numbered chapter folders in all path placeholders:

```text
CHAPTER_INDEX_PATH = content/2bac-pc-svt/01-limites-continuite/_index.md
CHAPTER_DIR = content/2bac-pc-svt/01-limites-continuite
MINI_LESSON_FILE = content/2bac-pc-svt/01-limites-continuite/lessons/lc-lesson-001.md
EXERCISE_FILE = content/2bac-pc-svt/01-limites-continuite/exercises/lc-ex-001.md
```
